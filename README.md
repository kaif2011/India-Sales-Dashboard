# India-Sales-Dashboard
A dynamic and interactive data visualization dashboard built to analyze sales performance across India—focusing on regional comparisons, product categories, customer segments, and overall business insights.

## 📌 Short Description / Purpose

The India Sales Performance Dashboard is a visually engaging and analytical Power BI report designed to analyze sales performance across different states and customer segments in India. 
The dashboard focuses on key business metrics such as total sales, profit, quantity, and customer distribution, along with insights into product categories, regional performance, and monthly sales trends. This tool helps businesses and analysts understand performance patterns and make data-driven decisions.

## 🛠️ Tech Stack

The dashboard was built using the following tools and technologies:
- 📊 Power BI Desktop – Used to create interactive dashboards and visualizations  
- 📂 Power Query – Used for data cleaning and transformation  
- 🧠 DAX (Data Analysis Expressions) – Used for calculated measures like total sales, profit, and KPIs  
- 🔗 Data Modeling – Used to create relationships between tables for better analysis  
- 📁 File Format – .pbix for development and .png for dashboard preview.

## 📁 Data Source
The dataset used in this project was provided as part of training. It contains structured sales data including details such as sales amount, profit, quantity, customer segments, product categories, and state-wise performance across India.

## 🚀 Features / Highlights

### 🔸 Business Problem:

Businesses often face difficulty in tracking sales performance across different regions, product categories, and customer segments. Identifying high-performing areas and understanding trends becomes challenging when working with raw data.

---

### 🔸 Goal of the Dashboard:

To build an interactive dashboard that:
- Tracks overall sales, profit, and quantity  
- Compares performance across states and regions  
- Analyzes customer segments and product categories  
- Helps in making data-driven business decisions.

### 🔸 Business Impact & Insights

- Helps identify top-performing states and product categories
